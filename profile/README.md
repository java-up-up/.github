集成了目前最为成熟的 SpringAI，并支持 Ollama、OpenAI、DeepSeek、阿里百炼 多种 AI 模型。并通过 MCP、Function Calling 与大麦项目进行联动，可以让 AI 智能的帮助用户执行操作。实现 只需聊天就能全部搞定！
